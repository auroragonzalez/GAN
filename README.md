# GAN

## Overview
This is the code inspired from [this](https://youtu.be/yz6dNf7X7SA) video on Youtube by Siraj Raval. We'll use a WGAN to create new images from ArrowHeads dataset heatmaps. 

## Dependencies (pip install) 
```
cv2
tensorflow( >=1.0)
scipy
numpy
```
## Usage
```
cd GAN_AU
python resize.py
python RGBA2RGB.py
python GAN.py
```
